### 已知 pattern / 学到的
- MiniMax-M2.7 的 Chinese \\uXXXX escape typo 规律：
  * 相邻 code point 混位（寒 U+5BD2 → 宽 U+5BBD）
  * 字符级 dropout 再补 digit（纪 U+7EAA → 谛 U+8C1B + "79"）
  * 偏旁错字（净 → 洁 / 财 → 贜 / 亏 → 贜）
  * 语义邻近替换（核心 → 校虚 / 催化剂 → 筹划）
- UTF-8 literal 原字存源码是最稳的 (cn-lexicon.js 路径已证)
